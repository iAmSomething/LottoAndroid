# 권한/개인정보 안내문 초안

## 1. 권한 사용 목적

### CAMERA
- 사용 위치: QR 스캔 화면
- 목적: 로또 복권 QR 코드 인식
- 미허용 시 동작: 카메라 스캔 기능 비활성화, 수동 URL 입력은 사용 가능

### POST_NOTIFICATIONS
- 사용 위치: 알림 설정 및 WorkManager 알림 노출
- 목적: 구매/결과 리마인더 알림 표시
- 미허용 시 동작: 알림 예약 저장은 가능하나 알림 표시 불가

### RECEIVE_BOOT_COMPLETED
- 사용 위치: 기기 부팅 완료 시점
- 목적: 기존 알림 예약 자동 복구
- 사용자 입력 데이터 접근: 없음

## 2. 데이터 처리 원칙
- 로컬 저장: 구매번호, 회차, 알림 설정, 통계 계산용 데이터
- 외부 전송:
  - 당첨 결과 조회용 공개 API 요청
  - QR URL 파싱 처리(기기 내)
- 서버 운영: 자체 백엔드 없음

## 3. 수집/공유 고지(스토어 입력용)
- 개인식별정보 수집: 없음
- 제3자 공유: 없음
- 광고/분석 SDK: 없음
- 필수 네트워크 사용: 당첨번호 조회 시점

## 4. 앱 내 고지 문구(초안)
```
매주로또는 구매번호와 설정 정보를 기기에 저장합니다.
카메라 권한은 복권 QR 스캔에만 사용되며, 알림 권한은 구매/결과 알림 표시를 위해 사용됩니다.
개인정보를 서버에 업로드하지 않습니다.
```
