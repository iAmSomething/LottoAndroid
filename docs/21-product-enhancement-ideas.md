# 제품 고도화 아이디어 보드 (v2)

## 1. 목적
- MVP 이후 사용자 리텐션, 사용 빈도, 품질 안정성을 동시에 끌어올릴 아이디어를 체계적으로 관리한다.
- 본 문서는 "아이디어 풀"이며, 실제 착수 우선순위는 `22-prioritization-matrix.md`를 기준으로 한다.

## 2. 고도화 방향
- 루틴 강화: 매주 반복 사용 동기를 강화한다.
- 의사결정 지원: 단순 번호 저장에서 "행동 추천"으로 확장한다.
- 플랫폼 확장: 모바일 중심에서 Wear OS까지 루틴 접점을 확장한다.
- 운영 안정화: 장애 감지/복구 시간과 회귀 리스크를 낮춘다.
