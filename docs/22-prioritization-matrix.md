# 고도화 우선순위 매트릭스 (2026-Q2)

## 1. 점수 규칙
- Impact(1~5): 사용자 가치/리텐션 영향
- Effort(1~5): 개발/QA 난이도
- Risk(1~5): 기술/운영/정책 리스크
- Priority Score = `(Impact * 2) - Effort - Risk`

## 3. 분기 실행 권고
- Sprint 1~2: `P0` 집중(D01, B02, B03, A02, G01, G06)
