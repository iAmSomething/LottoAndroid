# E2E 사용자 플로우

## 1. 홈 진입
1. 앱 실행
2. 홈에서 이번 회차, D-day, 주요 CTA 확인
3. QR 스캔 또는 번호 생성으로 이동

## 2. 번호 생성 플로우
1. A~E 5게임 자동 생성
2. 번호 터치로 잠금 처리
3. 필요 시 수동 숫자 입력(반자동 전환)
4. 입력 오류(빈값/범위/잠금충돌) 시 필드 하단 상세 오류 확인
5. 잠금 제외 랜덤 재생성
6. 이번 주 번호 저장
7. 동일 회차 자동생성 저장 시 최신 번호로 갱신(덮어쓰기)

## 3. 구매 등록 플로우
1. QR 스캔 또는 생성 번호 저장
2. 회차 기준으로 TicketBundle 생성
3. 스캔 실패 누적 시 재시도 가이드 카드 확인 후 재촬영 또는 수동 입력 전환
4. 홈 "이번 주 구매 번호" 카드에서 즉시 확인

## 3-1. 번호관리(Manage) 플로우
1. 하단 탭 `번호관리` 진입
2. 탭(이번주/보관함/스캔내역) 전환
3. 편집 모드에서 다중 선택 후 삭제
4. `+` FAB에서 번호 직접 추가/QR 스캔/가져오기 진입
5. 티켓 카드 클릭 시 상세 화면 확인

## 4. 당첨 확인 플로우
1. 결과 화면 진입
2. 최신 회차 API 조회(실패 시 로컬 캐시)
3. 저장 번호 자동 채점
4. 게임별 1~5등/낙첨 표시 + 일치 번호 하이라이트

## 5. 위젯 플로우
- 타입 A: 이번 주 번호 요약 + QR 진입
- 타입 B: 최신 당첨번호 + 내 결과 요약
