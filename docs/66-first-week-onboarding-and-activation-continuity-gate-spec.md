# 첫 주 온보딩/활성화 연속성 게이트 스펙 (S37)

## 1. 목적
- 설치 직후 7일 내 이탈을 줄이고 핵심 가치 체감 시간을 단축한다.
- 온보딩과 실제 사용 흐름 사이의 단절을 제거해 초기 리텐션을 높인다.

## 2. 적용 범위
- 대상 기간: 설치 후 D0~D7
- 대상 화면/흐름: `Home 첫 진입`, `번호 생성`, `결과 확인`, `번호 관리`, `알림 설정`, `Wear handoff`
- 대상 이벤트: 최초 진입, 첫 성공 경험, 복귀 사용자 재활성화, 미완료 플로우 재개

## 3. S37 Scorecard (O1~O5)
### O1. 첫 가치 도달 시간
- 확인 항목
  - 첫 진입 후 핵심 기능(번호 생성/결과 확인) 도달 경로가 명확하다.
  - 첫 성공 경험까지 필요한 탭 수/시간이 목표 범위 내다.
- PASS 기준
  - 첫 가치 도달 실패 이슈 0건

### O2. 온보딩-실사용 연속성
- 확인 항목
  - 안내 문구와 실제 UI 동작이 일치한다.
  - 온보딩 종료 후 바로 다음 행동 CTA가 자연스럽게 이어진다.
- PASS 기준
  - 온보딩 후 동선 단절 0건

### O3. 초기 개인화 적합성
- 확인 항목
  - 사용자 선호/상태에 따라 초기 추천(알림, 조회, 저장)이 과도하지 않게 제시된다.
  - 불필요한 설정 강제가 없다.
- PASS 기준
  - 초기 개인화 과잉/오작동 0건

### O4. 재방문/미완료 재개
- 확인 항목
  - D1~D7 재방문 시 이전 미완료 작업 재개 경로가 제공된다.
  - 재개 시 상태 유실 없이 바로 이어서 진행 가능하다.
- PASS 기준
  - 재방문 재개 실패 0건

### O5. 접근성/성능 동등성
- 확인 항목
  - 스크린리더/폰트스케일 1.3x/Reduce Motion에서도 온보딩 의미가 유지된다.
  - 초기 화면 로딩/전환이 저사양 환경에서도 임계치 이내다.
- PASS 기준
  - 접근성/성능 조건별 온보딩 회귀 0건

## 4. 개선 우선순위
1. 첫 가치 도달 경로 단순화(O1)
2. 온보딩-실사용 연결 고정(O2)
3. 초기 개인화 노출 강도 최적화(O3)
4. 재방문/재개 동선 강화(O4)
5. 접근성/성능 동등성 검증(O5)

## 5. 예외처리/안정성/성능 연동
- 네트워크 지연/오프라인
  - 온보딩 중 핵심 메시지와 fallback 경로를 유지한다.
- 세션 중단/복귀
  - 진행 단계와 입력 상태를 저장하고 재진입 시 재개한다.
- 저사양 장치
  - 첫 주 핵심 플로우에서 모션/로딩 부담을 축소한다.
